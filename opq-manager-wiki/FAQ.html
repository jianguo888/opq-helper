                # FAQ | 常见问题 🌈

                # Web面板登录报错

                你是否使用127.0.0.1来登录面板，如果是请使用localhost来登录

                # Setu姬如何使用

                • 在你第一次使用时，后台会弹出一个链接让你登录Pixiv，你点开链接登录即可
                • 登录后会是白色的页面，此时使用f12,找到一个pixiv://xxxxxxxx?code=xxxxx的请求将其中的code=xxxxxx私发给机器人，机器人后台会提示登录成功
                • 如果在登录过程中出现无法访问Pixiv的情况，此时你需要设置配置文件中的pixivproxy来出墙，如果你听不懂，请直接使用国外的VPS，感谢❤️

                # 机器人怎么面板和后台都在报错？

                请确保config配置文件中的OPQ配置正确，后台有提示[INFO] 连接服务器成功,如果有但后台还是报错请联系我

                # 关于本程序的内存占用问题

                在载入WordCloud(词云)模块后，程序会开一个单独的进程处理消息和分词，内存使用会达到200MB左右，内存紧缺的机器请悉知 😦 如果需要禁用词云模块，请在配置文件中设置

                disablemodule:
    - 词云生成

                # 关于数据库支持的类型

                目前支持sqlite3,mysql 如果需要支持其他的数据库请反馈

                # 奥运信息查询为啥报错？

                这个功能需要使用到docker

                docker pull idocking/go-wkhtmltox:latest
docker run -it -d -p 8080:8080 idocking/go-wkhtmltox:latest ./go-wkhtmltox run

                然后在config中

                htmltoimgurl: http://127.0.0.1:8080/v1/convert

                # 如何禁用模块

                在配置文件中的disablemodule配置项，配置如下

                disablemodule:
    - Setu姬
