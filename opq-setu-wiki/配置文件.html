首次启动需要配置的文件有:

OPQ-SetuBot/botoy.json (opens new window)

OPQ-SetuBot/plugins/bot_Setu/.PixivToken.json (opens new window)

OPQ-SetuBot/plugins/bot_Search_Picture/config.json (opens new window)

还需要运行OPQ-SetuBot/script下的generateConfig.py文件来给Setu插件生成各个群的配置文件 (opens new window)

# OPQ-SetuBot/botoy.json

字段名 说明
host OPQ的IP,本机运行就 http://127.0.0.1
port OPQ的端口,没修改过就是8888
bot bot的qq
superAdmin 你的qq
proxies_http http代理 Advanced Usage - HTTPX (python-httpx.org) (opens new window)
proxies_socks socks代理 例子: socks5://username:password@127.0.0.1:8888
group_blacklist 略
friend_blacklist 略
blocked_users 略
webhook 略
webhook_post_url 略
webhook_timeout 略

# OPQ-SetuBot/plugins/bot_Setu/.PixivToken.json

按照这里操作Pixiv OAuth Flow (with Selenium) (github.com) (opens new window)(往下拉,有中文)

最后把这里面斜体加粗的字符串完整粘贴到 .PixivToken.json 文件里的对应位置

❯ python3 pixiv_auth.py login

[INFO] Get code: 3s3Xc075wd7njPLJBXgXc4qS-...

refresh_token: uXooTT7xz9v4mflnZqJ...

expires_in: 3600

# OPQ-SetuBot/plugins/bot_Search_Picture/config.json

字段名 说明
APIKEY saucenao的apikey,申请地址: SauceNAO User Config (opens new window),注册后左侧api白字点进去的api key
