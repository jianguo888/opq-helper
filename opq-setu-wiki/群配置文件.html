路径 : /plugins/bot_Setu/dataBase/DB/configs/群号.json

修改实时生效,各个群独立

# setting

字段名 类型 说明
setu bool 开启关闭setu,群聊和群临时会话可分别控制
api bool 可以选择开关哪些API
r18 bool 是否开启r18,开启后默认返回porn和sexy混合的内容,可以通过[r18,R18]关键字来指定porn级别的色图,例:'来两张原神 黑丝 r18色图';porn关键字必须是最后一个tag
freq-limitCount int 在freq-refreshTime时间内限制调用setu功能的次数,默认120s限制调用10次,改成0就是无限制
freq-refreshTime int ↑↑↑↑↑↑↑↑ 同上, [refreshTime,limitCount]任意一个改成0就是无限制
quality str 可选项: [original, large, medium] ; bot发送的图片质量,默认中等画质,original就是发送原图
at bool bot发送setu时@调用的人
singleMaximum int 就是限制单次可以获取的最大数量,不建议太大,会炸
revokeTime int setu发出后的撤回时间,改成0就不撤回

# setuInfoShow

用于控制发送的消息中的setu信息

字段名 类型 说明
title bool 图片标题
picID bool 图片的P站ID
picWebUrl bool 图片的P站链接
page bool 图片所在画廊的第几P
author bool 作者
authorID bool 作者P站ID
authorWebUrl bool 作者P站链接
picOriginalUrl bool 图片原图
tags bool 图片的标签

# replyMsg

触发一些情况bot返回的消息

如果要修改的话,{aaa}这种字段不能修改,可以调整位置,但是不能少

字段名 类型 说明
inputError str 输入数量错误,例:'来asdasd张色图'
notFound str 你的xp太怪了,一张对的上的图都没有
tooMuch str 要的数量超过了限制(singleMaximum)的数量,例:'来999999张色图'
tooSmall str 数量小于1,例:'来-1张色图'
closed str setu功能关闭了
noR18 str 没有开启r18
insufficient str 没有获取到足够数量的setu
freqLimit str 触发频率限制了
