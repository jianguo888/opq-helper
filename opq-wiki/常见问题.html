# 常见问题

本页面记录了一些常见问题

# 功能限制

Windows 所有平台: 不支持长消息 不支持私聊 不支持语音 不支持图片 不支持JSON／XML 不支持添加好友 不支持好友/群处理

Linux 386 amd64平台: 不支持长消息 不支持JSON／XML

未列出的平台无限制

# 不能发消息

api返回无异常,但是bot没有发消息出去

可能原因:

  1. 异地登陆
  2. 发送频率过高
  3. QQ等级过低

屏蔽时间不确定几分钟或者几天 期间无法解决

json或者xml消息: 这2种信息会因为提交参数的不同或者属于铭感类而被屏蔽.

# 登陆不了

有可能服务器爆炸了,也可能是网络问题.

# 多账号

制作插件,提交PR,反馈bug,完善wiki等,可以找我PY多账号登录

# 控制bot的好友申请和群聊邀请

OPQBOT的插件目录(Plugins)自带了一个插件(Events.lua)可以简单处理,进去修改对应的参数就行了,如果不需要处理也可以直接删掉这个插件....

也提供了对应的API,可以自定义
