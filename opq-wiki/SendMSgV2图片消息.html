# 发送图片消息 群/好友/私聊

URL: http://127.0.0.1:8888/v1/LuaApiCaller

method :POST

# WebAPi请求参数:

params:

{
  'qq':123456, #机器人QQ号
  'funcname':SendMsgV2, #调用方法类型
}
# 通过网络URL发送图片:

发送好友图片消息 body(json):

{"ToUserUid":123456789,"SendToType":1,"SendMsgType":"PicMsg","PicUrl":"http://gchat.qpic.cn/gchatpic_new/304980169/636617867-2534335053-8E6B948D1E7A4F96DB5F9C4A6050FB02/0?vuin=123456789&term=255&pictype=0"}

发送群图片消息 body(json):


{"ToUserUid":123456789,"SendToType":2,"SendMsgType":"PicMsg","PicUrl":"http://gchat.qpic.cn/gchatpic_new/304980169/636617867-2534335053-8E6B948D1E7A4F96DB5F9C4A6050FB02/0?vuin=123456789&term=255&pictype=0"}

发送私聊图片 body(json):


{"ToUserUid":987654321 ,"GroupID":123456789,"SendToType":3,"SendMsgType":"PicMsg","PicUrl":"http://gchat.qpic.cn/gchatpic_new/304980169/636617867-2534335053-8E6B948D1E7A4F96DB5F9C4A6050FB02/0?vuin=123456789&term=255&pictype=0"}

# ToUserUid 好友QQ/群ID/私聊对象QQ
# GroupID 私聊对象所在的群ID
# SendToType  1 为好友消息 2 发送群消息  3 发送私聊消息
# PicUrl 有效的网络图片地址
# 通过本地发送图片:

发送好友图片消息 body(json):

{"ToUserUid":123456789,"SendToType":1,"SendMsgType":"PicMsg","PicPath":"./demo.jpg"}

发送群图片消息 body(json):


{"ToUserUid":123456789,"SendToType":2,"SendMsgType":"PicMsg","PicPath":"./demo.jpg"}

发送私聊图片 body(json):


{"ToUserUid":987654321 ,"GroupID":123456789,"SendToType":3,"SendMsgType":"PicMsg","PicPath":"./demo.jpg"}

# PicPath 本地路图片路径
# 通过图片Base64编码发送:

发送好友图片消息 body(json):

{"ToUserUid":123456789,"SendToType":1,"SendMsgType":"PicMsg","PicBase64Buf":"xxxx"}

发送群图片消息 body(json):


{"ToUserUid":123456789,"SendToType":2,"SendMsgType":"PicMsg","PicBase64Buf":"xxx"}

发送私聊图片 body(json):


{"ToUserUid":987654321 ,"GroupID":123456789,"SendToType":3,"SendMsgType":"PicMsg","PicBase64Buf":"xxxx"}
# 通过MD5发送图片(可多图):

仅支持群 发送群图片消息 通过上传图片接口取图片信息或者通过收到的图片消息进行整合 body(json):

{
  "ToUserUid":987654321,
  "SendToType":2,
  "Content":"哈哈",
  "SendMsgType":"PicMsg",
  "PicMd5s":[
    {
      "FileId":2826722046,
      "PicMd5":"VcI+GuRJgguHVwTqrlqk8w==",
      "PicSize":8861
    },
    {
      "FileId":2801670139,
      "PicMd5":"5BFzQoQ+D3qruPEVm+i7HQ==",
      "PicSize":40944
    },
    {
      "FileId":2380188012,
      "PicMd5":"MtZQ9ax9CMtwF4npjFverQ==",
      "PicSize":65846
    }
  ]
}

⚠️ 可根据自己需要选择是通过网络Url发送还是本地发送 PicMd5s PicPath PicUrl PicBase64Buf 字段 同时存在时 处理优先级依次是 PicMd5s PicUrl PicPath PicBase64Buf 发送多图仅支持群的md5图片

发送群闪照 body(json):


{"ToUserUid":123456789,"SendToType":2,"SendMsgType":"PicMsg","PicPath":"./demo.jpg","FlashPic":true}

# FlashPic true 为发送闪照 默认false

发送图文消息

{"ToUserUid":757360354,"SendToType":2,"SendMsgType":"PicMsg","PicPath":"./demo.jpg","Content":"HelloWorld","FlashPic":true}

#只需增加一个Content字段即可 图文的顺序问题会在消息宏中介绍

--通过宏[秀图40001] 范围 40000-40006 实现秀图发送 群有效好友无效 后续消息宏介绍
